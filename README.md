# AndroidCharts
Android Charts base on ECharts

## Usage:

1. Add it in your root build.gradle at the end of repositories:

```
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
```

2. Add the dependency:

```
dependencies {
	compile 'com.github.frendyxzc:Charts:v0.0.1'
}
```
